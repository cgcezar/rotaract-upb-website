ASSETS FOLDER — what to put here
=================================

This site currently uses CSS-drawn graphics and gradients, so it works with
ZERO image files. But you can make it your own by adding real photos and the
official logos.

1) OFFICIAL ROTARY / ROTARACT LOGOS  (recommended)
   The Rotary wheel and the word "Rotaract" are trademarks of Rotary
   International. Do NOT recreate them by hand. Download the correct, official,
   approved logo files (free for members) from the Rotary Brand Center:

        https://brandcenter.rotary.org

   Look for the "Rotaract" lockup and your club-name version. Save the file
   here as, for example:  rotaract-logo.svg

   Then, in index.html, you can replace the small <span class="nav__mark"> in
   the header with:  <img src="assets/rotaract-logo.svg" alt="Rotaract" class="nav__logo" />

2) PHOTOS FOR FLAGSHIP EVENTS  (optional, looks great)
   Add JPGs/PNGs of your projects here, e.g.  event-polio.jpg
   To use a photo as an event banner, see CONTENT-GUIDE.md → "Adding a photo
   to an event."

3) FAVICON
   favicon.svg (already included) is the little icon in the browser tab.
   Replace it with your own if you like — keep the same filename.

Tip: compress photos before uploading (https://squoosh.app is free) so the
site stays fast. Aim for under ~300 KB per image.
